== Description ==
This plugin allows admin to view orders on orders listing page rather than going to order detail page.There will be a link called quick view which will appear on hovering the order title in the order listing page.Admin can also view multiple orders using bulk actions.

= Note =
This plugin is ready to use with any theme.

== Installation ==

 Quick Setup


 1) Install plugin through backend of your Wordpress site or via FTP.
 

 Having trouble with installation? Just check our website with free videos on installation and setup.

 From Wordpress Backend (automatic):
 Go to `Plugins -> Add new` -> click on Upload Plugin and upload the File woocommerce-quick-order-view.zip,
 or by Search Plugins, type in:`woocommerce-quick-order-view` and click on ìnstall`

 Manual Setup:
 1. Upload `woocommerce-quick-order-view` Folder from unzip woocommerce-quick-order-view.zip to the `/wp-content/plugins/` directory
 2. Activate the plugin through the 'Plugins' menu in WordPress
